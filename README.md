## Project Title

Side Quest Week 1: VS Code & GitHub Setup

---

## Description

This Side Quest lets students learn about repositories, set up Visual Studio Code,
and create a basic p5.js sketch that is published using GitHub Pages.

---

## Interaction Instructions

- Open the GitHub Pages link to view the sketch.
- The sketch runs automatically on load.
- No user interaction is required for this activity.

---

## Assets

No external assets were used for this side quest.  
The `assets/` folder is included with extra subfolders: images, sounds, fonts.
